**Overview:**\
This game is going to be an otome style game. There will be three main character types: crown prince/princess, grand duke/duchess, magic tower master; the player will be able to make choices and level up into the game. The game will be based on a fantasy college experience and the player will take exams while interacting with the main characters. Challenges and puzzles will be put into different exams and the player will have to complete these puzzles to graduate. When graduating the player’s most affectionate character will be their ending partner. The game will be successful when you graduate, however if the player's affection levels for main characters fall below zero then the player will lose. 

**Theme:** Fantasy, college - Otome game

**Required features:**\
-Three main characters change gender depending on player\
-Storyline\
-Exams with challenges and puzzles (5)\
-Affection levels\
-Losing if affection level falls below zero\
-Ending lines\
-Winning if you graduate\
-Leveling up (fighting people, etc.)\
-World to move around in (grid) between events\
-Fighting\
-Tutorial(for all puzzles)

**Desired features:**\
-Different endings based on past decisions you’ve made\
-Two character same affection level - battle\
-AI storyline that tells the same story in a different way each time\
-Add fighting in school \
-Achievement page/summary\
-Boosts for main character partnership

**Target audience:**\
-Young people\
-People who like beating games and strategy, challenges\
-People who like connections with charcters
